# Shortcut reference

## {% icon 'doggo' %} In-game

| Keys                                                                                                                                   | Action                            |
| -------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------- |
| {% key '←' %}{% key '↑' %}{% key '↓' %}{% key '→' %} or {% key 'W' %}{% key 'A' %}{% key 'S' %}{% key 'D' %} or {% key 'Left Mouse' %} | Move character {% icon 'doggo' %} |
| {% key 'Space' %} or {% key 'Enter' %} or {% key 'Left Mouse' %}                                                                       | Interact                          |

## {% icon 'pencil' %} Editor

### 3D controls

| Keys                                                     | Action               |
| -------------------------------------------------------- | -------------------- |
| {% key 'Left Mouse' %}                                   | Place block / sprite |
| {% key 'Right Mouse' %}                                  | Delete block         |
| {% key 'Middle Mouse' %}                                 | Rotate view          |
| {% key 'Shift' %} + {% key 'Middle Mouse' %}             | Move view            |
| {% key 'Mouse Wheel' %}                                  | Zoom view            |
| {% key 'Shift' %} (hold)                                 | Layer draw mode      |
| {% key 'Shift' %} (hold) + {% key 'E' %} / {% key 'Q' %} | Go up / down layer   |

### Shortcuts

| Keys                             | Action                     |
| -------------------------------- | -------------------------- |
| {% key 'Tab' %}                  | Toggle block / sprite mode |
| {% key 'F' %}                    | Toggle block / ramp        |
| {% key 'R' %}                    | Rotate ramp                |
| {% key 'Ctrl' %} + {% key 'S' %} | Save                       |
